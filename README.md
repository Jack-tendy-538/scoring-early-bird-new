# score_early_bird

This project is also available in [English](README_en.md).

这是一款专为**个人或小型团队**设计的桌面端习惯追踪与打卡激励工具。它通过自动化的积分奖励和定时提醒，将枯燥的重复行为变为有趣的成长游戏，帮助你轻松建立并坚持好习惯。

---

### **它解决了什么痛点？**

1.  **动力不足，难以坚持**：单纯靠意志力打卡很容易半途而废。本项目引入**游戏化积分机制**，为连续打卡设置看得见的奖励，提供持续的正向反馈。
2.  **容易忘记，缺乏提醒**：忙起来总会忘记打卡。工具内置**智能定时提醒**功能，在预设时间弹出窗口，确保你不会错过任何一次记录。
3.  **记录零散，不便回顾**：手工记录分散在不同地方，难以总结。程序能自动生成**清晰美观的Markdown格式周报/月报**，让你一目了然地回顾自己的坚持历程。
4.  **工具臃肿，设置复杂**：很多习惯类App功能繁多但用不上。本工具**极度轻量、配置灵活**，一个简单的YAML配置文件就能自定义所有规则，无需复杂操作。

---

### **采用的技术栈**

*   **核心开发语言**： `Python 3` - 确保良好的跨平台兼容性和丰富的库支持。
*   **用户界面（GUI）**： `Tkinter` + `ttkbootstrap` - 使用Python标准GUI库搭配现代化主题，打造出类似Windows 11风格的简洁桌面界面，无需浏览器即可运行。
*   **配置管理**： `YAML` - 采用人类易读的配置文件格式，所有规则（如积分、提醒时间、显示设置）都可轻松修改，无需改动代码。

---

### **支持的核心功能**

1.  **可配置化积分系统**：
    *   支持设置不同连续打卡周期（如3天、7天）的奖励积分，激励长期坚持。
2.  **智能定时提醒**：
    *   可自由开启/关闭，并精确设置上、下午的提醒时间，防止遗忘。
3.  **个性化名称列表**：
    *   允许用户为打卡项自定义有趣的“花名”，增加打卡的趣味性和归属感。
4.  **数据可视化输出**：
    *   自动将打卡数据生成为排版优美的Markdown表格文档，便于分享、打印或存档。
5.  **一站式设置管理**：
    *   提供图形化设置窗口，可对积分规则、提醒时间、界面字体、输出格式等进行集中管理。

---

### **适用的典型场景**

*   **学生自我管理**：用于记录每日早起、背单词、运动打卡，通过积分激励自己。
*   **团队目标共建**：小型团队或学习小组内部，用于同步追踪项目进度、每日学习打卡，并自动生成团队打卡周报。
*   **个人习惯养成**：无论是希望培养每天阅读、喝水、练字的习惯，还是希望戒除某些不良习惯，都可以通过此工具进行可视化的自我监督。
*   **轻量级数据记录**：替代纸质或简单的Excel表格，用于需要定期、重复记录某项数据的场景，且希望自动化生成统计视图。

**总结来说**，`score_early_bird` 就像一位安静、可靠的数字伙伴。它没有复杂社交功能，专注解决“坚持记录”和“获得激励”这两个核心问题，通过高度的自定义性，成为真正贴合你个人习惯的轻量化管理工具。
